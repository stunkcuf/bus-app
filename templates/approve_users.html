About the Approval Process

When new drivers register for the system, they need manager approval before they can access their dashboard. This ensures only authorized personnel can manage routes and students.

  • Approve: Grants the user access to the driver dashboard and all driver features
  • Reject: Denies access and removes the user from the pending list

Approved drivers will receive the standard "driver" role. For manager access, use the User Management page after approval.
