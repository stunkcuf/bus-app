How to Use Report Builder

1. Select Data Source

Choose from available data sources like Fleet Data, Routes, Students, etc.

2. Choose Fields

Select the fields you want to include in your report from the available list.

3. Apply Filters

Add filters to narrow down your data (optional). Click "Add Filter" to begin.

4. Generate & Export

Click "Generate Report" to view results. Export to CSV, Excel, or PDF formats.
